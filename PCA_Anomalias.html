Detección de Anomalías¶

El enfoque de deteccin de anomalas (tambin conocido como valores atpicos o extraos), basado en el anlisis de componentes principales (PCA) se ha utilizado con xito para monitorear sistemas con variables altamente correlacionadas.

En este notebook se propone un esquema de deteccin de anomalas basado en PCA, sobre un conjunto de datos que representan imgenes de perros y gatos, con la intencin de revelar si una imagen particular es un gato o no. Probaremos eventualmente tambin el caso contrario.

}

El anlisis de componentes principales buscar abstraer las caractersticas mas relevantes que describen a un gato (o perro). Para lograr una buena abstraccin, es necesario normalizar los datos, ya que cada columna (variable o caractrstica) de nuestros datos, pueden estar definidas entre rangos de valores muy distintos, donde las variables con mayor rango de variabilidad lograran opacar a las dems variables.

Con la siguiente lnea se obtienen los errores asociados a la re-proyeccin del paso anterior. Si los residuales son pequeos, significa que la imgen corresponde a un gato, en otro caso sera un perro u otra cosa (asumimos perro puesto que la base unicamente tiene perros y gatos)

Queremos validar cmo se ven afectadas las proyecciones, o lo que es lo mismo, cmo incrementan los residuales, cuando usando menos componentes principales. Vemos a continuacin la evaluacin de los residuales para todos los conjuntos de datos usando entre 2 y 70 componentes.

componentes <- 2:ncol(pc$x)

pc_residuales <- data.frame(N_CP=NULL, gatos.test.res=NULL, perros.train.res=NULL, perros.test.res=NULL)
for(i in componentes){
     pc_residuales[i-1, 1] <- i
     
     # gatos test
     gatos.test.resid <- gatos.test - ( (gatos.test %*% matriz.proyec[,1:i]) %*% t(matriz.proyec[,1:i]) )
     pc_residuales[i-1, 2] <- mean(apply(gatos.test.resid, 1, norm, type="2"), na.rm=T)
     
     # perros train
     perros.test.resid <- perros.test - ( (perros.test %*% matriz.proyec[,1:i]) %*% t(matriz.proyec[,1:i]) )
     pc_residuales[i-1, 3] <- mean(apply(perros.test.resid, 1, norm, type="2"), na.rm=T)
     
     # perros test
     perros.train.resid <- perros.train - ( (perros.train %*% matriz.proyec[,1:i]) %*% t(matriz.proyec[,1:i]) )
     pc_residuales[i-1, 4] <- mean(apply(perros.train.resid, 1, norm, type="2"), na.rm=T)
}

Clasificación de perros y gatos¶

Bucamos ahora un algoritmo que nos permita clasficar perros y gatos. Para ello aplicamos componentes principales sobre la base de entrenamiento para perros, luego realizamos proyeccin sobre los datos de prueba y aplicamos el algoritmo k-medias, esperando poder clasificar correctamente las imagenes.

In [29]:
centro.perros <- colMeans(perros.train)

gatos.train <- gatos.train -
               matrix(centro.perros, nrow(gatos.train), ncol(gatos.train), byrow=TRUE)
gatos.test <- gatos.test -
              matrix(centro.perros, nrow(gatos.test), ncol(gatos.test), byrow=TRUE)
perros.train <- perros.train -
                matrix(centro.perros, nrow(perros.train), ncol(perros.train), byrow=TRUE)
perros.test <- perros.test -
               matrix(centro.perros, nrow(perros.test), ncol(perros.test), byrow=TRUE)

## Componentes principales
pc_perros <- prcomp(perros.train)
matriz.proyec <- pc_perros$rotation

## Proyecciones
gatos.test.py <- (gatos.test %*% matriz.proyec) %*% t(matriz.proyec)

perros.train.py <- (perros.train %*% matriz.proyec) %*% t(matriz.proyec)

total.py <- rbind(gatos.test.py, perros.test.py)
py_label <- c(rep(1, 29), rep(2, 29))
